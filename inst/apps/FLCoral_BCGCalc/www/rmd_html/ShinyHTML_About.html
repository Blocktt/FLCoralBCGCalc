Background

This website calculates Biological Condition Gradient (BCG) scores for coral samples reefs in Florida (Figure 1). The USEPA has published a practitioner’s guide to BCG development that would be useful introductory reading for someone new to the BCG framework (USEPA 2016). A report on the development of the Florida Coral BCG models is forthcoming.

Want to try running the calculator but don’t have input files? Download the Florida BCG Test File .
